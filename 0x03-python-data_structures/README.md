# Python - Data Structures: Lists, Tuples

## Introduction
This repository focuses on Python scripts and a C program dedicated to manipulating lists and tuples. Each file serves a unique purpose in enhancing your understanding of these fundamental data structures.

## Execution
Before running the scripts, ensure you've granted execution permissions to the necessary files. For instance, use the following command:
```bash
chmod u+x 12-switch.py
```
This step is crucial for the checker to successfully validate the scripts.
